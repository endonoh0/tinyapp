# TinyApp Project

TinyApp is a simple URL shortener, build in Node and Express.

## Dependencies

- Node.js
- Express
- EJS
- bcrypt
- body-parser
- cookie-session
- method-overide

## Installation

#### Step 1:
Install all dependencies
```npm install```

#### Step 2:
Run the development web server
```npm run start```

#### Step 3:

Visit http://localhost:8080/register to create a new account and shorten your first URL!
